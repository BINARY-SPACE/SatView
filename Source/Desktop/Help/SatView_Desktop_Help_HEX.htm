Hexadecimal Display (HEX)

SUMMARY

Saturday, October 10, 2015
Version 1.0

A Hexadecimal Display (HEX) is capable to monitor raw (unprocessed) telemetry data as well as Memory Dumps. In addition to that, so called Spy Locations extract selected portions of the data for further investigation.

How do I...

Open a HEX Display

  1. Push the HEX button inside the Telemetry Displays panel of the Displays ribbon bar category.
    This opens the HEX Open Dialog Box.
  2. Choose between the available display types by checking the appropriate radio button in the dialog box:

    Type

    Description

    Normal

    Shows the raw data belonging to a specified telemetry packet.
    Select the format to be viewed in the Display combo box.
    Note:
    When monitoring telemetry data that is not conform with the CCSDS Packet Telemetry Standard this option displays the complete telemetry formats generated by the spacecraft.

    Stream

    Shows the raw data of every telemetry packet (or format) received.
    Note:
    This option is not applicable when the telemetry data is not compatible with the CCSDS Packet Telemetry Standard.

    Dumps

    Displays the content of a specified memory dump.
    Select the memory dump to be viewed in the Display combo box.
  3. Choose the data mode by either checking the Realtime or History radio button.
    Follow the instructions about how to retrieve with a HEX display when requesting historical telemetry data.
  4. Optionally configure the HEX display.
  5. Push the OK button.

Configure the Settings

Many aspects of a HEX display are configurable. Layout, retrieval source and print jobs can be configured and modified as required. Furthermore, it is also possible to define areas of special interest, called Spy Locations, that show selected bits extracted from the telemetry data stream.

  1. Open the HEX Settings Dialog Box if it is not open yet:
    Activate the HEX display and select the Settings... item on the application button's menu. Alternatively, push the HEX button on the display's toolbar.

  2. Read more about:

    Modifying the Layout

    Specifying the Appearance:

    1. Select the Layout page.

    2. Changing the Number of Bytes per Line

      • Inside the Format group box adjust the number in the For unformatted data use... edit field.
        Note:
        This option does not apply to the Dumps display type.

      Changing the Color of the Data

      1. Specify the kind of data those color should be changed by selecting it in the For kind of data use... combo box.
        Note:
        This option does not apply to the Dumps display type.
      2. Select the desired color in the corresponding ...color combo box.

      Enumerating Lines and Columns

      • Mark either the Enumerate lines, or Enumerate columns check box or both depending on the kind of enumeration required and adjust the numbers in the corresponding edit fields.
        Note:
        This option does not apply to the Dumps display type.

      Changing the Fonts

      1. Inside the Fonts group box select the Title, Data, Spy Header or Spy Locations radio button depending on which font should be changed. The Title radio button refers to the title bar on top of each display, the Data to the displayed information and the Spy Header/Locations to the locations of special interest, when defined, at the bottom part of the display.
      2. Select the font name in the Name combo box and select the desired style and size in the Style combo box and Size edit field respectively. A sample of the chosen font is always displayed in the Sample area.

    Controlling the Operating Flags:

    1. Select the Layout page.

    2. Bad Data Flag

      Every telemetry parameter has an associated data quality flag referring to its value. By default, a parameter value with a bad quality flag is not updated.

      • Mark the Bad Data check box if the telemetry parameter values should be displayed even if their data quality flag is set to bad.

    Using Spy Locations

    1. Select the Contents page.

    2. Adding a new Spy Location

      1. Specify the radix with which the value at the spy location should be displayed by checking the appropriate radio button:

        Radix

        Description

        Decimal

        The value composed of the bits found at the spy location is displayed with a decimal radix. The first bit is always interpreted as the most significant bit.

        Binary

        The extracted value is displayed with a binary radix.

        Octal

        The extracted value is displayed with a octal radix.

        Hexadecimal

        The extracted value is displayed with a hexadecimal radix.
      2. Specify the position of the spy location by filling in the Byte, Bit and Length edit field.
        Valid ranges for the various edit fields are:

        Field

        Valid Range

        Byte

        >= 0

        Bit

        0...7

        Length

        1...32
        Note:
        The value of a spy location is not updated if its position is (partly) outside the telemetry data packet (or format).
      3. Enter the name of the spy location into the Description edit field.
      4. Determine the color x with which the parts of the spy location should be displayed by selecting the appropriate options in the For part of spy location use color x combo boxes.
      5. Push the Add button.
      Note:

      This option does not apply to the Dumps display type.

      Modifying a Spy Location

      1. Select the spy location to be modified in the Spy Locations list box and perform the required changes.
      2. Push the Modify button.
      Note:

      This option does not apply to the Dumps display type.

      Deleting a Spy Location

      1. Select the spy location to be deleted in the Spy Locations list box.
      2. Push the Remove button in order to delete the selected spy location. Use the Remove All button to delete all at once.
      Note:

      This option does not apply to the Dumps display type.

    Specifying the Retrieval Source

    1. Select the Retrieving page.
    2. Select the retrieval file to become the source for any retrieval requests by highlighting it in the Data Source combo box.
      Push the Inspect button to check a potential retrieval file. The details about the inspected retrieval file are shown in the Source Information group box.
      Note:
      The set of retrieval files listed in the Data Source combo box can be specified in the Retrievals page of the System Settings Dialog Box.

    Using Print Job Profiles

    1. Select the Printing page.

    2. Creating a new Print Job Profile

      1. Type the name of the print job profile to be created into the Name combo box.
        Note:
        The Default print job profile is used when printing with the quick printing method and can only be modified but not removed.
      2. In the Properties group box specify the print profile to be used with this print job profile by selecting the corresponding profile in the Print Profile combo box. The print profile specifies the printer and paper related properties whereas the print job profile adds some, more HEX display specific, printing directives.
      3. Specify how many lines of the telemetry data or of the spy locations should be printed by checking either the Print all lines or Print the first/last n lines radio button and by filling-in the corresponding edit fields.
        Note:
        Mark the Print Spy Locations check box if the spy locations should be printed as well. Printing restrictions for spy locations can only be specified when this option is checked.
      4. Type a comment into the Text edit field if additional information should be added to the title of the print-out. Mark the Clear after use check box if this text should be used in a single print-out only.
      5. Push the Add button.

      Modifying a Print Job Profile

      • Select the name of the print job profile to be modified in the Name combo box and perform the necessary changes (see 'Creating a new Print Job Profile').

      Deleting a Print Job Profile

      1. Select the name of the print job profile to be deleted in the Name combo box.
      2. Push the Remove button to remove the selected print job profile. Use the Remove All button to delete all existing profiles.
        Note:
        The Default print job profile cannot be deleted.

    Scheduling Print Jobs

    1. Select the Jobs page.
    2. Select the print job to be scheduled in the Defined list box.
    3. Specify when the print job should be executed by marking one or more of the available options:

      Option

      Description

      Print at YYYY.DDD.hh:mm:ss

      Prints the HEX display at a specified time.
      Type the time into the associated time edit field.

      Print at hh:mm:ss every day

      Prints the HEX display every day at a specified time.
      Type the time into the associated time edit field.

      Print at name event

      Prints the HEX display when the event name is in a signaled state.
      Select the event in the associated combo box.
    4. When printing at a certain time is enabled it is possible to specify how many times the printing should be repeated and with what interval. In the For repetition type combo box select the repetition characteristics for either daily or non-daily printing. In the case of non-daily printing specify the interval in minutes after which another printing should occur by checking the Repeat every n minutes and by typing the interval n into the associated edit field.
      For both kinds of printing stop conditions can be specified by selecting either the Stop after n copies or Stop at time YYYY.DDD.mm:hh:ss radio button and by filling in the appropriate data into the edit fields.
    5. Push the > button to activate the print job. The name of the print job is copied from the Defined list box into the Active one in order to indicate the activation.
      Whenever a print job needs to be deactivated but not deleted select it in the Active list box and then push the < button.
      Note:
      Deactivated print jobs are not scheduled.
  3. Push the OK button.

Perform Retrievals

SatView™ provides a convenient way to retrieve telemetry data. Navigating through historical telemetry data is a simple process thanks to the various playback modes and high capacity random access storage devices.

  1. Open the HEX Retrievals Dialog Box by selecting the Retrievals... item on the application button's menu when the display is activated. Alternatively, push the Retrievals button on the display's toolbar.
  2. In the Retrieve from Time edit field inside the Historical Data group box specify the time at which the retrieval should start.
    The syntax for the specification of the time is: YYYY.DDD.hh:mm:ss.
    Note:
    Mark the Exact Time check box in order to force the retrieval to start exactly at the specified time. This is done automatically for the Stream type of displays. For the other types it depends on the way how telemetry data is processed.
  3. Specify when the retrieval should be stopped by checking one of the following options:

    Option

    Description

    Stop at YYYY.DDD.hh:mm:ss

    Stops the retrieval at the specified time. Type that time into the associated time edit field.

    Stop when no more data is available

    Stops retrieving telemetry data when no more data is available in the retrieval file.

    Switch to realtime

    Automatically switches to real-time when no more telemetry data is available.
  4. Specify whether or not memory dumps should be generated from the historical telemetry data by marking the Redo Dumps check box. Selecting the Archiving check box as well saves the memory dumps, enabling the Notifications check box sends them via FTP to clients. The way how this is performed is defined through the Memory Dumps Service.
    Note:
    The options described in this point are available for the Dumps display type only.
  5. Determine the playback mode by checking one of the following possibilities:

    Option

    Description

    Automatic

    Replays the telemetry data with a specified speed.
    Adjust the speed with the Speed slider.
    Note:
    The playback mode may be changed interactively during the retrieval. Read below how to navigate through the historical telemetry data.

    Pseudo Realtime

    The telemetry data is replayed at a speed close to real-time.
    Note:
    The playback mode may be changed interactively during the retrieval. Read below how to navigate through the historical telemetry data.

    Manual

    The telemetry data closest to the specified starting time is monitored and then the retrieval is stopped.
    Note:
    The playback mode may be changed interactively during the retrieval. Read below how to navigate through the historical telemetry data.
  6. Specify an interval in minutes if a sequential retrieval is not required. This option is useful when navigating through the retrieval file and searching for a certain incident.
    Note:
    Depending on the telemetry data standard used it is not always possible to retrieve with the interval specified i.e. the de-facto interval may be smaller or bigger than the one specified for the Generic and obviously also for the Dumps display type.
  7. Push the OK button.
Note:

Push the Realtime Mode check box to terminate any retrievals and to switch back to real-time.

Find Displays

SatView™ provides a convenient way to search for a HEX display representing a specified telemetry packet or for a display which has a title containing a certain word or words. The search criteria can contain boolean operators.

  1. Open the HEX Find Dialog Box if it is not open yet:
    Push the Find... button on the HEX Open Dialog Box.
  2. Select the search key in the Key combo box.
  3. Select a search value expression in the Value combo box or enter one into the same edit field (if no choices are available).
    Note:
    The search values may also contain the boolean operators '+' (AND) and '|' (OR). Whenever one or more of these operators are specified the operands need to be set into double quotes. There is no operator precedence; the search value expression is just evaluated from the left to the right.
    Examples:

    Key

    Search Value

    Action

    Packet

    "001R"

    Searches for the HEX display that corresponds to the telemetry packet '001R'.

    Description

    "AOCS" + "DYNAMICS"

    Searches for all HEX displays that contain the words 'AOCS' and 'DYNAMICS' in their title.
  4. Push the Search button.
  5. Select the HEX display to be opened in the Results list box and push the Open button or double-click on the display to be opened.
    Note:
    Push the Close button in order to discontinue the search for a HEX display.

Print Displays

  1. Activate the HEX display to be printed.
  2. Choose one of the following printing methods:

    Method

    Description

    How to do it

    Quick Printing

    This method is the most simple one because it does not need any further interaction of the user in order to print a HEX display. The way how the display is printed is defined by the Default print job profile.

    Push the Print button on the display's toolbar.

    Interactive Printing

    Whenever a HEX display needs to be printed in a different way than defined by the Default print job profile this method is the one to be used. It allows the modification of the print properties without changing the profile.
    • Select the Print... item on the application button's menu when the display is activated.
    • Change the settings in the Print Dialog Box as required.
    • Push the OK button.

    Scheduled Printing
    Event-driven Printing

    Sometimes it may be advantageous to print a HEX display automatically at a certain time or when a predefined telemetry event occurs. This can be achieved by scheduling a print job.
    • Select the Settings... item on the application button's menu when the display is activated or push the Settings button on the display's toolbar.
    • Select the Print page in the HEX Settings Dialog Box.
    • Add a print profile as required.
    • Push the OK button.

Prerequisites:
The privilege 'Using hexadecimal display windows' must be enabled in order to use any HEX displays.
